## ✨ 主な機能

- **マルチコンテンツ表示**:
  - Webサイト、動画、画像を自由に追加・配置。
  - URL入力と、ローカルファイルのドラッグ＆ドロップに対応。
- **柔軟なレイアウト**:
  - 列数やコンテンツ間の余白をスライダーで直感的に調整。
  - ボタン一つで最適なレイアウトに自動整形。
- **一括コントロール**:
  - 全動画の再生、停止、ミュート、音量均一化をワンクリックで実行。
- **個別コントロール**:
  - コンテンツを個別に選択し、リロード、複製、削除、回転、反転が可能。
  - 動画の場合は、音量や再生速度も個別に調整できます。
- **特殊モード**:
  - **主軸モード**: 選択したコンテンツを大きく表示し、他をサムネイルとして並べます。
  - **没入モード**: UIを隠してコンテンツに集中できます。
- **保存 & 共有**:
  - **共有URL**: 現在のレイアウトをURLとして出力し、他の人と共有できます。
  - **プリセット**: よく使うURLの組み合わせをブラウザに保存・呼び出し。

## 🚀 使い方

1.  **コンテンツ追加**: 左サイドバーからURLを入力、またはファイルをドラッグ＆ドロップします。
2.  **レイアウト調整**: 右サイドバーのスライダーで列数や余白を調整します。
3.  **個別操作**: 中央のコンテンツをクリックで選択し、右サイドバーでURL変更や削除などの操作をします。

## ⚠️ 注意事項

- **共有とプリセット**: ローカルファイル（ドラッグ＆ドロップで追加したもの）は共有URLやプリセットには保存されません。Web上のURLのみが対象です。
- **表示制限**: サイト側のセキュリティ設定（`X-Frame-Options`など）により、一部のWebページは表示できない場合があります。
